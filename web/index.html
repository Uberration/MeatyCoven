What is Coven?

A local-first runtime substrate for visible coding agents.

Coven is a single Rust daemon that owns harness PTYs, session state, and an append-only event ledger on your machine. CastCodes is the primary workspace and public proof surface for that runtime. The coven CLI/TUI remains the operator surface for installing, checking, and managing the daemon directly.

  • Local-first. The daemon, store, and socket live under $COVEN_HOME.
  • Harness-neutral. Codex and Claude Code run through the same supervised PTY layer.
  • Project-rooted. Every launch pins and revalidates a canonical project root.
  • Inspectable. Sessions and events are SQLite rows you can browse, replay, archive, summon, or sacrifice.

Quick start

Install, check the environment, start the daemon, then launch a session.

01

Install Coven

npm install -g @opencoven/cli
02

Check your environment

coven doctor

doctor reports whether supported harnesses are on PATH and what to install next.

03

Start the daemon

coven daemon start
coven daemon status
04

Launch your first session

cd /path/to/your/project
coven run codex "describe this repo"

Session browser

Browse live and archived work without losing the ledger.

coven sessions opens a human-friendly browser of every live and archived session. Rejoin a PTY, view the log, restore an archived session, archive finished work, or permanently sacrifice non-running sessions with explicit confirmation.
